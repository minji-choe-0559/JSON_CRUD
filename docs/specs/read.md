# Spec: Read

## 개요
- JSON 파일에 저장된 연락처(Record) 전체 목록을 조회하거나,
  특정 `id`로 검색하는 기능.

## 입력
- 파일 경로 (`std::string`).
- 검색 시: `id` (`int`).

## 출력
- 전체 목록: `std::vector<Record>`.
- id 검색: 찾으면 해당 `Record`를 담은 `std::optional<Record>`,
  못 찾으면 `std::nullopt`.

## 엣지 케이스 / 예외 상황
- 파일이 없거나 비어있으면 (JsonStore가 빈 배열로 처리) 전체 목록은
  빈 벡터를 반환한다.
- 존재하지 않는 id로 검색해도 예외를 던지지 않고 `std::nullopt`를
  반환한다 (CLAUDE.md 에러 처리 원칙: 존재하지 않는 id 조회는
  정상적인 흐름으로 처리).
