# Spec: Delete

## 개요
- `id`로 연락처를 찾아 삭제하고 파일에 반영하는 기능.
- 삭제 전 확인(y/n) 프롬프트는 콘솔 UI 관심사이므로 이 기능의
  책임이 아니다 (CLAUDE.md: 콘솔 입출력은 메뉴 루프에만 둔다).
  이 기능은 "삭제할 대상을 찾아 지우고, 지운 레코드를 반환"하는
  순수 데이터 로직만 담당한다. 확인 프롬프트는 이 반환값을 이용해
  콘솔 메뉴 쪽에서 구현한다.

## 입력
- 파일 경로 (`std::string`), 삭제할 `id` (`int`).

## 출력
- 삭제 성공 시 삭제된 `Record`를 담은 `std::optional<Record>`
  (콘솔에서 "무엇을 지웠는지" 보여줄 때 사용).
- 해당 `id`가 없으면 `std::nullopt` (예외 아님).

## 엣지 케이스 / 예외 상황
- 존재하지 않는 `id`로 Delete 시도 시 예외를 던지지 않고
  `std::nullopt`를 반환한다 (CLAUDE.md 에러 처리 원칙).
- 삭제 대상이 아닌 다른 레코드는 그대로 유지된다.
- 레코드가 하나뿐인 파일에서 그 레코드를 삭제하면 빈 배열이 저장된다.
