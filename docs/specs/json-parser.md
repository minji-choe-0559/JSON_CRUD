# Spec: JsonParser

## 개요
- JSON 형식 문자열을 파싱해 `JsonValue`로 변환하는 기능.

## 입력
- JSON 형식의 `std::string`.
  - null, true/false, 숫자(정수/실수/음수), 문자열, 배열, 객체를 포함.
  - 배열/객체는 중첩될 수 있다.
  - 값 사이 공백(스페이스/탭/개행)은 무시한다.

## 출력
- 정상 파싱 시 해당 JSON 구조를 그대로 담은 `JsonValue` 반환.

## 엣지 케이스 / 예외 상황
- 빈 문자열, 공백만 있는 문자열 파싱 시 예외.
- 괄호/대괄호 불일치, 콤마 누락, 따옴표 누락 등 문법 오류 시 예외.
- 문자열 안의 이스케이프 시퀀스(`\"`, `\\`, `\n` 등)는 지원한다.
- 지원하지 않는 이스케이프 시퀀스는 예외.
- 숫자 뒤/앞에 불필요한 문자가 붙는 경우 예외.
- 파싱 후 문자열이 남아있는데 값은 이미 끝난 경우(예: `1 2`) 예외.
