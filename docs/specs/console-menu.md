# Spec: 콘솔 메뉴 에러 처리

## 개요
- `AIProject5.cpp`의 메뉴 루프(`runConsoleMenu`)가 사용자의 비정상 입력이나
  JSON 파일 처리 중 발생하는 예외 상황에서도 프로그램을 종료하지 않고
  안내 메시지를 출력한 뒤 메뉴로 돌아오도록 하는 동작을 정의한다.

## 입력
- 메뉴 선택 번호(1~6), 각 기능별 후속 입력(이름/전화번호/id 등).

## 출력
- 정상 처리 시 각 기능(Create/Read/Update/Delete)의 기존 출력.
- 비정상 상황에서는 에러 안내 메시지를 출력하고 메뉴 루프를 계속한다.

## 엣지 케이스 / 예외 상황
- 메뉴 선택 입력에 숫자가 아닌 값을 입력한 경우: `cin` 실패 상태를
  `clear()`+`ignore()`로 복구하고 "잘못된 입력입니다" 안내 후 메뉴를 다시
  보여준다. 표준입력이 EOF(더 이상 입력 없음)에 도달한 경우에만 루프를
  종료한다.
- `data.json` 쓰기/읽기 중 예외(`std::runtime_error` 등)가 발생한 경우
  (예: 파일 열기 실패, 파싱 실패): 예외를 잡아 에러 메시지를 출력하고
  프로그램을 종료하지 않은 채 메뉴로 돌아온다.

## 테스트 케이스 목록

  안내 후 메뉴가 다시 표시됨
- [x] 예외 케이스: 표준입력이 EOF에 도달하면 메뉴 루프가 정상 종료됨
- [x] 예외 케이스: Create/Update/Delete 처리 중 예외 발생 시 크래시 없이
  에러 메시지 출력 후 메뉴로 복귀

콘솔 입출력 로직은 CLAUDE.md 규칙에 따라 `main.cpp`에만 위치하고
별도 모듈로 분리하지 않으므로, 위 테스트 케이스는 GoogleTest 자동화
대신 빌드 후 실제 실행(`echo`로 표준입력 주입)으로 수동 검증한다.
